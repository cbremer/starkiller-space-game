# High-Fidelity Concept Lab

This folder stores planning-phase visual direction material for the next-fidelity pass of Starkiller.

## Contents

- `inspiration/README.md` — archived notes from the user-provided reference images.
- `samples/` — generated concept assets (SVG) for rapid prototyping in-engine.
  - background concepts
  - sprite-sheet concept blocks
  - tileset atlas concept blocks
  - `manifest.json` index of generated files

## Regeneration

You can regenerate the planning samples with:

```bash
python scripts/generate_high_fidelity_concept_svgs.py
```

## Notes

- These are brainstorming artifacts and are **not final production art**.
- Samples are intentionally stylized toward readable gameplay silhouettes while exploring richer presentation.
