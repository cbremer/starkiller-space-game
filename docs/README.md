# Docs

All project documentation lives under `docs/`.

## Projects

- `project-starkiller-prototype/` - Current Godot prototype docs.
- `art-direction/` - Art direction inspiration board, concept asset manifest, and worktree plan.

## Naming Convention

Files inside a project folder use:

`YYYY-MM-DD-<type>-<detail>.md`

Types:
- `brainstorm` - discovery notes and reference research
- `spec` - implementation spec and behavior contracts
- `tasks` - session-level implementation checklists

When starting a new working session:
1. Add a new `tasks` file with the next session number/date.
2. Update that project's `README.md` timeline table.
