## Naming Convention

For new docs in this folder, use `YYYY-MM-DD-<type>-<detail>.md`.

- `brainstorm`: pre-build exploration and evidence gathering
- `spec`: product and implementation contract
- `tasks`: session-level implementation plan/checklist

The existing files `inspiration-board.md` and `worktree-plan.md` are legacy docs that intentionally keep short names and are explicitly exempt from the date-based naming rule above.
